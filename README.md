  Orden-Arte es una aplicacion diseñada para ayudar a grupos artisticos con mantener ordenada la informacion sobre sus actividades.
  Por un lado, la app le permitira a un usuario crear cuantos grupos desée, dentro de los cuales tendra la posibilidad de tener una carpeta por cancion para agregar en ella
las partituras de cada instrumento. Así mismo, en cada carpeta también se podrá agregar archivos multimedia para, por ejemplo, subir audios de referencia para entender mejor la musica.
  Por otro lado, la app tambien brindará la posibilidad de manejar la informacion de los acontecimientos que interpelen a cada grupo (ensayos, grabaciones, shows en vivo, etc). 
En este apartado se podra incluir, por ejemplo, informacion sobre la localidad y horarios de cada acontecimiento, info sobre el dinero recaudado o sobre que vestimenta se debe usar o sobre 
que tipo de evento sera.
